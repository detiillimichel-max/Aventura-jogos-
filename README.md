# 🏃‍♂️ Git Commit Runner

> Corra, desvie dos bugs e colete commits! Um endless runner estilo Subway Surfers com tema GitHub.

### **🎮 Jogar Agora**
**https://SEUUSER.github.io/git-commit-runner/**

Instale como PWA direto no celular: "Adicionar à tela inicial"

### **📱 Preview**
Deslize para os lados para mudar de lane. Toque rápido para pular. Desvie dos 🐛 bugs e colete 📝 commits para bater seu recorde.

### **✨ Features**
- **100% PWA**: Funciona offline após o primeiro load
- **Mobile First**: Controles por swipe otimizados pra celular
- **GitHub Theme**: Personagem com hoodie do GitHub e spray "COMMIT"
- **Endless Runner**: Velocidade aumenta com o tempo
- **Highscore**: Recorde salvo no localStorage
- **Sem dependências**: Vanilla JS puro, roda em qualquer browser
